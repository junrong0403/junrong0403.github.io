各式API開發

   負責處理各式API開發，讓APP可以處理複雜的動作或是需要和資料庫存取的狀況或使用排程定時觸發需定時處理的操作等。
  • 1. Firebase Cloud Messaging個別推播
  • 2. 股票webstock連接
  • 3. Sign in with Apple
  • 4. 財政部發票條碼相關
  • 5. 股票新聞自動抓取

1. Firebase Cloud Messaging個別推播

   透過分別處理各APP各裝置的FCM_token將其開發成API供APP上傳到DB存下，後端封裝成model觸發，處理各式各樣的操作， 像存錢通知、自訂股票通知、口罩地圖通知、訂單完成通知...等。

2. 股票webstock連接

   使用node-red圖形化操作，以stomp.js操作webstock供app連接接收股票即時價格，並利用FCM立即推播通知使用者即時的股價。

3. Sign in with Apple

   串接Apple登入，這次採用JWT的方式，以PHP接收redirect的code，再與POST到AppleAPI確認取得本次授權的email， 開發過程中由於PHP沒有完整的套件或是加密的格式需管換的問題(.p8 .pem)、JWK轉換成公鑰、Base64(url safe)等處理上花了許多的時間，做後也成功完成串接。

4. 財政部發票條碼相關

   開發以PHP模擬財政部登入、註冊API嵌入APP，讓使用者可直接在APP裡完成操作。發票操作上之接利用財政部API， 可自動排程同步發票資訊到DB中，並利用FCM立即推播通知使用者。

5. 股票新聞自動抓取

   使用node-red圖形化操作第三方API，將各股票新聞正規化的存入資料庫中，並再開發API讓APP取得指定股號之新聞。
